# Elecciones2026

Para proyectar los resultados a partir de la data en bruto de todos los timestamps (si no se provee parametro):

```bash
python proyectar_resultados.py --n-donors 3 [--timestamp 20260414_0302]
```

Para generar el mapa web de imputaciones de todos los timestamps (si no se provee parametro):

```bash
python mapa_imputaciones.py [--timestamp 20260414_030W2]
```

Para generar el grafico web de evolucion:

```bash
python grafico_evolucion.py --top 5
```
